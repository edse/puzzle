HTML5 Image Puzzle

Learning HTML5 by doing!

We've built a HTML5 puzzle game for learn the following HTML5 features:

  • Canvas
  • Image manipulation
  • Application Cache API (Offline browsing)
  • Audio API
  • Touch support

Canvas, image manipulation, and RequestAnimationFrame

The game runs on canvas. It loads an image and slice it in a few pieces. The RequestAnimationFrame provide power efficient and smooth animations.

Audio, and fullscreen APIs

Background music and sound effects using the Audio API. Also fullscreen mode support.

Application Cache/Offline browsing support

You can keep playing even when you go offline.
